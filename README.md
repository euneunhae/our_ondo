## 2. 디자인 컨셉

### 2-1. 핵심 키워드

따뜻함 · 아기자기함 · 세련됨 · 손그림 감성

### 2-2. 색상 팔레트

| 역할 | 색상명 | HEX 코드 |
|------|--------|----------|
| 메인 포인트 | 주황 | `#E8742A` |
| 서브 포인트 | 고동 | `#7B3F2E` |
| 어센트 | 연두 | `#8DB87A` |
| 배경 | 크림 베이지 | `#FDF8F3` |
| 본문 텍스트 | 따뜻한 다크 | `#2E1F18` |

### 2-3. 디자인 필수 요소

- **크레파스/색연필 느낌의 선 (필수):** 섹션 구분선, 테두리, 장식 요소에 손그림 질감의 SVG 선을 사용할 것. 직선이 아닌 약간 불규칙하고 두툼한 질감의 선으로 표현.
- **폰트:** 제목은 손글씨 계열 (예: Nanum Pen Script), 본문은 가독성 있는 서체 (예: Noto Sans KR)
- **모서리:** 딱딱한 직각보다 둥근 모서리(border-radius) 사용
- **전체 톤:** 아기자기하되 과하지 않은 세련된 감성

## 4. 페이지별 상세 내용

### 4-1. 홈 (메인 페이지)

- 홈페이지 이름 "우리의 온도" 타이틀 표시
- 히어로 이미지: 클라이언트 제공 가족 사진 1장으로 교체 예정 *(사진 별도 전달)*
- 메인 화면 화분 이미지 없음 (제거)
- 최근 일상 포스팅 3개 미리보기 (카드형)
- 각 메뉴로 이동하는 링크 버튼

### 4-2. 우리의 일상

상단에 두 개의 카테고리 탭으로 구분하여 표시.

#### ① As Time goes by
- 우리 가족의 실질적인 일상 기록 공간
- 포스팅 1건 구성:
  - 날짜
  - 사진 1~3장
  - 제목
  - 본문 (짧은 글, 3~5줄)
  - 태그

#### ② Momentary
- 엄마(운영자)의 감성을 담는 개인 공간
- 일상에서 순간순간 포착한 사진 + 짧은 감성 글 위주
- 포스팅 1건 구성:
  - 날짜
  - 사진 1장 (감성 포착 이미지)
  - 짧은 감성 텍스트 (1~3줄, 필수 아님)
  - 태그

### 4-3. 우리의 여행

- 분기별 여행 목록 (카드형)
- 여행 기록 1건 구성:
  - 여행지 이름 / 날짜
  - 대표 사진 1장
  - 여행 일자별 사진 + 내용
  - 아이들 한마디 (자유 입력)

### 4-4. 우리의 성장

- 자녀별 탭 또는 섹션으로 구분 (첫째 / 둘째)
- 성장 기록 1건 구성:
  - 기록 날짜 (월)
  - 사진 1~2장
  - 자유 텍스트 (근황, 첫 경험, 한마디 등)

## 6. 기술 스택 제안

> 운영자가 직접 글을 올릴 수 있어야 하므로, **CMS(콘텐츠 관리 시스템)** 기반으로 구성하는 것을 권장합니다.

| 구분 | 추천 옵션 | 이유 |
|------|-----------|------|
| CMS | WordPress / Webflow / 직접 개발 | 비개발자도 쉽게 글 작성 가능 |
| 호스팅 | Cloudflare Pages / Vercel / Cafe24 | 안정적, 저비용 |
| 이미지 저장 | Cloudinary / 서버 로컬 | 사진 관리 편의성 |

## 7. 콘텐츠 운영 계획 (참고용)

개발자가 관리 도구를 설계할 때 참고하세요.

| 콘텐츠 | 업로드 주기 |
|--------|------------|
| 일상 포스팅 | 주 2~3회 |
| 여행 기록 | 분기 1회 (연 4회) |
| 성장일기 | 월 1회 |
| 가족 소개 | 최초 1회 (이후 드물게 수정) |

## 8. 참고 사항

- 운영자는 코딩 경험이 없으므로, **글 작성 화면은 최대한 단순하게** 설계해 주세요.
- 사진 업로드 시 자동 압축/리사이징 처리를 권장합니다.
- 아이 사진이 포함되므로 **검색 엔진 노출 차단(robots.txt)** 설정을 기본값으로 요청합니다.
- 모바일 사용 비중이 높으므로 모바일 UX를 우선 고려해 주세요.
